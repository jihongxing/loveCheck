## 🐛 常见问题排查

### 问题1: 导航按钮不显示
**原因**: 国际化文本未加载
**解决**: 检查 `frontend/src/i18n/zh.js` 和 `en.js` 是否包含 `app.nav.companySearch` 和 `app.nav.companyReport`

### 问题2: 点击导航按钮无反应
**原因**: 组件未正确导入
**解决**: 检查 `App.vue` 是否导入了 `CompanySearch` 和 `CompanyReport` 组件

### 问题3: 提交举报后无响应
**原因**: 后端API未启动或路由配置错误
**解决**: 
1. 检查后端是否运行：`curl http://localhost:8080/api/v1/health`
2. 检查路由配置：`backend/cmd/lovecheck/main.go` 是否包含 `/api/v1/company/report`

### 问题4: 查询公司时报错
**原因**: 数据库表未创建
**解决**: 
1. 重启后端，自动执行AutoMigrate
2. 手动检查：`psql -U lovecheck -d lovecheck -c "\dt company*"`

### 问题5: 国际化文本显示为key
**原因**: 翻译文本缺失
**解决**: 检查 `i18n/zh.js` 和 `i18n/en.js` 是否包含完整的 `companyReport` 和 `companySearch` 对象

### 查看测试数据
```sql
-- 查看举报记录
SELECT id, display_name, industry, location_city, status, created_at 
FROM company_records 
ORDER BY created_at DESC 
LIMIT 5;

-- 查看公司统计

       COUNT(DISTINCT company_hash) as total_companies
```
