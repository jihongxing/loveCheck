# 电话号码区号联动功能说明

## 功能概述

当用户切换界面语言时，电话号码输入组件会自动切换到对应国家的区号。

## 语言到国家代码映射

| 语言 | 语言代码 | 国家 | 国家代码 | 区号 |
|------|---------|------|---------|------|
| 中文 | zh | 中国 | CN | +86 |
| 英语 | en | 美国 | US | +1 |
| 日语 | ja | 日本 | JP | +81 |
| 韩语 | ko | 韩国 | KR | +82 |
| 泰语 | th | 泰国 | TH | +66 |
| 越南语 | vi | 越南 | VN | +84 |
| 西班牙语 | es | 西班牙 | ES | +34 |
| 印地语 | hi | 印度 | IN | +91 |

## 实现细节

### 1. PhoneInput 组件
- 位置: `frontend/src/components/PhoneInput.vue`
- 功能: 提供国家选择器和电话号码输入
- 支持手动选择任何国家的区号

### 2. CompanyReport 组件集成
- 位置: `frontend/src/components/CompanyReport.vue`
- 实现:
  ```javascript
  // 语言到国家代码映射
  const langToCountry = {
    zh: 'CN',
    en: 'US',
    ja: 'JP',
    ko: 'KR',
    th: 'TH',
    vi: 'VN',
    es: 'ES',
    hi: 'IN'
  }
  
  // 监听语言变化，自动更新国家代码
  watch(locale, (newLocale) => {
    reporterCC.value = langToCountry[newLocale] || 'CN'
  })
  ```

### 3. 用户体验
- 用户切换语言时，电话输入框的国家区号会自动更新
- 用户仍然可以手动选择其他国家的区号
- 电话号码会在切换国家时自动清空（避免格式不匹配）

## 测试步骤

1. 打开公司举报页面
2. 观察电话号码输入框的默认区号（应该是当前语言对应的国家）
3. 切换语言（例如从中文切换到日语）
4. 验证区号是否自动从 +86 变为 +81
5. 尝试手动选择其他国家的区号
6. 再次切换语言，验证区号是否再次自动更新
